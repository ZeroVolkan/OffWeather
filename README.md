# Offweather app

## Run the app

### Uv package manager

Run as a desktop app:

```bash
uv run flet run
```

Run as a web app:

```bash
uv run flet run --web
```


## Build the app

```bash
flet build apk -v # Android

flet build ipa -v # iOS

flet build macos -v # macOS

flet build linux -v # Linux

flet build windows -v # Windows
```
